2.1.1 contrast($background, $type)

Return a color contrasted by parent block color

  • $background: 'light', 'light' or 'dark'
  • $type: 'primary', values possible in $contrast-colors

2.2.1 padding($level, $side)

Apply padding on a block

  • $level: 'normal', The level spacing defined in $block-padding.
  • $side: '', The side of block to apply padding. (if blank, apply for all sides)
